
# Business Website

## Overview
This is a **professional business website** template designed for small businesses, startups, and personal brands. It focuses on building a **strong online presence**, showcasing services, projects, and pricing. Built using **HTML, CSS, and JavaScript**, it is modern, fast, and responsive.

---

## Features
- Fully **responsive design** for mobile, tablet, and desktop  
- Clean and modern **UI/UX design**  
- Sections include **Home, About, Services, Projects, Pricing, Contact**  
- Call-to-action buttons for **lead generation**  
- SEO-friendly meta tags for better search visibility  
- Smooth scrolling and interactive elements  
- Pricing section to highlight services  

---

## Technologies Used
- **HTML5** – Structure of the website  
- **CSS3** – Styling and responsive layout  
- **JavaScript** – Interactive elements  
- **Fonts & Icons** – Google Fonts, Font Awesome
